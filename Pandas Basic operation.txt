KEY NOTE:
1. Index vs Column (MOST IMPORTANT)

.loc[] and .at[] work only on index labels, not column values.
===================================================
In pandas, label-based access (loc, at) works only on index labels, position-based access (iloc, iat) works on row/column numbers; if a value is in a column (like a name), either filter by that column or set it as the indexâ€”string matching is case-sensitive.
